## 🔥 **L'IA dévore notre planète**

### **La vérité que l'industrie tech cache :**
- 🤖 **ChatGPT consomme 10x plus** qu'une recherche Google
- 💧 **6,4 milliards de litres** d'eau évaporés par Microsoft en 2022
- 🔥 **550 tonnes de CO₂** pour entraîner GPT-3 (= 500 vols NY-SF)
- 📈 **×100 plus polluant** : GPT-4 vs GPT-3.5
- 🌍 **2050 : 37% de plus** que la consommation de la France entière

> *"Pendant que nous utilisons l'IA pour écrire nos emails, des centres de données géants consomment plus d'électricité que 92% des pays du monde."*

## ✨ **Fonctionnalités**

### 🎯 **Hero Section - Révélations Choquantes**
- **Statistiques en temps réel** avec sources vérifiables
- **Animations immersives** avec particules IA
- **Call-to-action percutants** avec navigation intelligente

### 📊 **Calculateur d'Impact IA Personnalisé**
- **5 types d'usage IA** : ChatGPT, Claude, images, code, vidéo
- **Facteurs d'émission réels** basés sur la recherche académique
- **Comparaisons parlantes** : km voiture, bouteilles d'eau, arbres
- **Export des résultats** en JSON + partage natif

### 🌱 **Solutions Concrètes par Profil**
- **👤 Utilisateurs** : Optimiser prompts, choisir bons modèles
- **🏢 Entreprises** : Politiques d'usage, modèles locaux, formation
- **💻 Développeurs** : Pruning, quantification, edge computing

### 🧠 **Quiz Éducatif Interactif**
- **7 questions chocs** sur l'impact réel de l'IA
- **Faits surprenants** : Croissance 4-5x par an, 1 image = 670 questions
- **Engagement citoyen** : Challenge 30 jours IA responsable

### 📈 **Section Impact - Données Factuelles**
- **3 catégories d'impact** : Énergie, Carbone, Eau
- **Sources académiques** : Polytechnique, AIE, universités
- **Coûts cachés** : Fabrication, entraînement, inférence, refroidissement

## 🚀 **Installation & Lancement**

### **Prérequis**
```bash
Node.js >= 18.0.0
npm >= 8.0.0
```

### **Installation rapide**
```bash
# Cloner le repository
git clone https://github.com/votre-username/hackathon-ia-for-good.git
cd hackathon-ia-for-good

# Installer les dépendances
npm install

# Lancer en développement
npm run dev

# Build pour production
npm run build

# Preview build
npm run preview
```

### **Scripts disponibles**
```bash
npm run dev      # Serveur développement avec hot reload
npm run build    # Build optimisé pour production
npm run preview  # Preview du build local
npm run lint     # Linting ESLint
```
